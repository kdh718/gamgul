# Gamgyul

과외용으로 만든 문제 생성기입니다
exe 파일을 다운로드 받아서 바로 실행할 수 있습니다
현재 최신 버전은 v1.0.1입니다

초6-중1 수준의 문제로 구성되어 있으며 현재 문제 유형은 21종입니다
추후 난이도와 범위를 선택할 수 있도록 업데이트할 예정입니다

문제 유형은 과외하면서 자주 나오는 유형을 모아서 직접 만든 문제이며
문제 생성 시 특정 교재로부터 복제하지 않았음을 명시합니다
